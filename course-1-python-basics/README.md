# Course 1: Python Basics & Command Line Magic

> **Audience:** Ages 11+ · **Python:** 3.12+ · **No prior coding required**  
> **Pace:** ~8–10 weeks at 1–2 lessons/week (~30–45 min each)

Learn to write Python scripts, use the terminal, make choices with logic, organize code with functions and collections, and draw with Turtle.

**Master curriculum:** [CURRICULUM.md](../../CURRICULUM.md)

## Exit skills

When you finish Block 5, you can:

- Run Python from the terminal and read error messages
- Use variables, `input()`, f-strings, and string methods
- Branch with `if` / `elif` / `else` and repeat with `for` / `while`
- Write functions, lists, and dictionaries
- Draw shapes and patterns with Turtle
