## Block 0 readiness checklist

Tick these before starting Block 1 — they're about *understanding*, not setup:

- [ ] I can explain, in my own words, what a programming language is
- [ ] I know a program is a list of exact instructions run in order
- [ ] I can describe the journey: code → interpreter → result
- [ ] I understand why order matters (top to bottom, line by line)

---

## For parents and teachers

**No setup needed:** Block 0 is pen-and-paper / read-aloud friendly. It builds the mental
model (instructions, interpreter, order) that prevents confusion once real coding starts in
Block 1. The practice tasks (writing plain-English "programs" and tracing output by hand)
are great to do together.
