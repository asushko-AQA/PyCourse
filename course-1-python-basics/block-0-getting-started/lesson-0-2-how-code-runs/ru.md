## Объяснение

В Уроке 0.1 ты узнал, что программа — это список точных инструкций. Но как написанные слова *делают* что-то на самом деле? Проследим путь от **твоего кода** к **настоящему результату на экране**.

Когда ты пишешь Python, ты сохраняешь его в файле с окончанием `.py` — это твой **исходный код** (рецепт, который ты написал). Сам по себе этот файл — просто текст. Чтобы он *заработал*, компьютер использует помощника — **интерпретатор Python**.

Интерпретатор — как **переводчик** между тобой и компьютером. Он читает твой Python **по одной строке, сверху вниз**, и говорит компьютеру, что именно делать для каждой строки. Компьютер делает это, и ты видишь **результат** — текст, рисунок или окно игры.

Путь всегда такой:

**твой код (файл `.py`)** → **интерпретатор Python (переводчик)** → **результат на экране**

### Шаг 1: Исходный код — это твой записанный рецепт

Ты печатаешь инструкции в файл вроде `hello.py`. Пока ничего не происходит — это просто сохранённый текст, который ждёт.

### Шаг 2: Интерпретатор — это переводчик

В Python есть интерпретатор (ты установишь его в Уроке 1.1). Когда ты «запускаешь» файл, интерпретатор просыпается и начинает читать твой код.

### Шаг 3: Строка за строкой, сверху вниз

Интерпретатор читает **строку 1**, выполняет её, потом **строку 2**, и так далее — по порядку. Поэтому **порядок важен**: поменяй две строки местами — и результат может измениться.

### Шаг 4: Появляется результат

Для каждого `print(...)` интерпретатор говорит компьютеру показать текст. Позже ты будешь делать рисунки и игры — но путь всегда тот же: код → интерпретатор → результат.

### Шаг 5: Взгляд на твою первую программу

Вот классическая первая программа, `hello.py`:

```python
print("Hello, world!")
```

Когда интерпретатор прочитает эту строку, компьютер покажет `Hello, world!`. Ты напечатаешь и запустишь это по-настоящему в Уроке 1.1.

## Пример кода

Первая программа часто всего одна-две строки — `hello.py`:

```python
print("Hello, world!")
print("My code is alive!")
```

Интерпретатор читает строку 1, показывает её текст, потом читает строку 2 и показывает её текст — всегда по порядку.

## Запуск кода

Чтобы запустить файл Python, ты наберёшь команду вроде этой (по-настоящему сделаешь это в Уроке 1.1):

```text
python hello.py
```

Интерпретатор Python читает файл сверху вниз, и компьютер показывает:

```text
Hello, world!
My code is alive!
```

(Пока ничего запускать не нужно — здесь мы лишь заглядываем, как «запуск» превращает код в результат.)

## Уголок отладки

**Проблема:** Программа печатает строки в «неправильном» порядке.

```python
print("Second")
print("First")
```

Вывод:

```text
Second
First
```

**Причина:** Интерпретатор выполняет строки **сверху вниз**, а не по смыслу слов. Он напечатал `Second` первым, потому что эта строка была первой.

**Исправление:** Расставь строки в том порядке, в котором хочешь их выполнить. Компьютер следует позиции, а не смыслу.
