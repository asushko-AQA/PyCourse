## For parents and teachers

**Saving files:** Use **Save** after every edit. Confirm the extension is `.py`, not `.py.txt`. In Windows Explorer, enable **View → Show → File name extensions**.

**Same folder, two views:** File Explorer and the terminal can show the same folder. Compare the path in the terminal prompt with the folder path in VS Code's title bar.

**Python not found:** If the terminal says `python is not recognized`, see the Debug Corner in [Lesson 1.1](lesson-1-1-installing-python/en.md) — usually **Add to PATH** was missed during install.
