## What you'll build / Что ты создашь

**EN:** A working Python setup, VS Code ready to code, and your first `hello.py` script running in the terminal.

## What you'll learn / Что ты узнаешь

- Download and install Python with **Add to PATH**
- Install VS Code and the Python extension
- Open the integrated terminal
- Create and run a first `hello.py` script
- Read `python --version` output
