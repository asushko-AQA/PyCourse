## Объяснение

Привет, Создатель! Сейчас ты установишь два мощных инструмента:

1. **Python** — волшебный переводчик. Ты пишешь команды понятным языком, а Python превращает их в действия компьютера.
2. **VS Code** — твой **Центр управления**. Здесь ты пишешь код, сохраняешь файлы и открываешь **Портал команд** (терминал).

Представь, что это игра:

- **Python** = игровой движок, который всё запускает.
- **Add to PATH** = ярлык на рабочем столе, чтобы запускать Python откуда угодно. **Не пропускай это!**
- **VS Code** = твоя база, где ты создаёшь заклинания (программы).
- **Терминал** = портал, куда ты вводишь команды и запускаешь код.

---

### Шаг 1: Скачай Python

1. Открой браузер.
2. Перейди на **https://www.python.org/downloads/**
3. Нажми большую жёлтую кнопку: **Download Python 3.12** (или новее).
4. Дождись загрузки файла. Например: `python-3.12.x-amd64.exe`.

Ты получил свой магический переводчик. Отличное начало!

---

### Шаг 2: Установи Python — и отметь «Add to PATH»!

Это шаг **с боссом**. Одна галочка решает всё.

1. Дважды щёлкни по скачанному файлу.
2. **ВАЖНО:** на первом экране посмотри вниз.
3. Поставь галочку: **«Add python.exe to PATH»** (или **«Add Python to PATH»**).
4. Нажми **«Install Now»**.
5. Подожди сообщение **«Setup was successful»**.
6. Нажми **Close**.

**Зачем нужен PATH:** PATH — это карта сокровищ для компьютера. Если Python в PATH, ты можешь писать `python` в терминале из любой папки. Без галочки компьютер скажет: «Python не найден» — и игра усложнится зря.

**Уже установил, но забыл PATH?** Запусти установщик снова, выбери **Modify** и включи **Add Python to environment variables**.

---

### Шаг 3: Проверь, что Python работает

1. Нажми **Win**, введи **cmd**, открой **Командную строку**  
   *(позже в VS Code: **Terminal → New Terminal**)*.
2. Введи и нажми **Enter**:

```text
python --version
```

3. Должно появиться что-то вроде:

```text
Python 3.12.x
```

Если видишь номер версии — **уровень пройден!** Python готов.

---

### Шаг 4: Установи VS Code

1. Перейди на **https://code.visualstudio.com/**
2. Нажми **Download for Windows**.
3. Запусти установщик. Настройки по умолчанию подходят.
4. Дождись конца установки.
5. Открой **Visual Studio Code**.

---

### Шаг 5: Добавь расширение Python (усиление)

1. В VS Code нажми **Extensions** слева (четыре квадрата).
2. Найди **Python**.
3. Установи **Python** от **Microsoft**.
4. Это расширение помогает VS Code понимать твой код — как умный помощник в Центре управления.

---

### Шаг 6: Открой папку проекта в VS Code

1. **File → Open Folder**.
2. Выбери папку курса (например: `Documents\PyCourse`).
3. Нажми **Select Folder**.

Теперь ты в своей кодинг-базе!

---

### Шаг 7: Открой терминал в VS Code

Терминал — твой **Портал команд**.

1. **Terminal → New Terminal** (или **Ctrl + `**).
2. Внизу откроется панель — это терминал.
3. Мигает курсор — портал открыт и ждёт команду.

**Совет:** можно нажать **+** в панели терминала, чтобы открыть ещё одну вкладку.

## Пример кода

Создай свой первый файл-заклинание. В VS Code:

1. **File → New Text File**.
2. Сохрани как **`hello.py`** в папке проекта.
3. Введи код ниже.

Файл **`.py`** — это Python-скрипт, свиток с инструкциями для компьютера.

**Файл: `hello.py`**

```python
# hello.py
# Your first Python spell — this file tells the computer what to say.

# print() sends a message to the screen (like opening a magic scroll).
print("Hello, World!")

# You can send more than one message. Each print() is a new line.
print("I am learning Python!")
print("My code is alive!")
```

**Что делает каждая часть:**

| Строка | Значение |
|--------|----------|
| Строки с `#` | Заметки для людей. Python их не выполняет. |
| `print("...")` | Показать текст на экране. |
| Кавычки `"..."` | Текстовая **строка** — цепочка символов, как бирка с именем. |

Готовый файл есть в папке урока: [starter/hello.py](starter/hello.py).

## Запуск кода

Пора произнести заклинание! Используй **терминал** (Портал команд).

### 1. Перейди в папку, где лежит `hello.py`

Если файл в `Documents\PyCourse`, введи:

```text
cd Documents\PyCourse
```

*(Измени путь, если папка в другом месте. `cd` — **c**hange **d**irectory, «перейти в папку».)*

### 2. Запусти скрипт

```text
python hello.py
```

Нажми **Enter**.

### Ожидаемый результат

```text
Hello, World!
I am learning Python!
My code is alive!
```

Если видишь три строки — **получилось!** Ты запустил настоящий Python-код.

### Дополнительная проверка

```text
python --version
```

---

### Запуск из VS Code (проще)

1. Открой `hello.py` в редакторе.
2. Открой терминал (**Terminal → New Terminal**).
3. Убедись, что ты в нужной папке (при необходимости `cd`).
4. Введи `python hello.py` и нажми **Enter**.

**Или:** нажми **Run** (иконка воспроизведения) справа вверху, если установлено расширение Python.

## Задание для практики

**Название квеста:** Представление Создателя

**Твоё задание (сделай сам):**

1. Создай файл **`my_intro.py`** (или измени `hello.py`).
2. С помощью **`print()`** выведи **четыре строки**:
   - Строка 1: `Hello, World!`
   - Строка 2: Твоё имя — например: `My name is Sam.`
   - Строка 3: Твой возраст — например: `I am 11 years old.`
   - Строка 4: Что ты хочешь создать кодом — например: `I will build a cool game!`
3. Сохрани файл.
4. Открой терминал и выполни:

```text
python my_intro.py
```

5. Сделай скриншот или расскажи другу, что появилось на экране.

**Бонусные звёзды (по желанию):**

- Запусти `python --version` и запиши номер в тетрадь. Это твой уровень Python.
- Добавь ещё одну строку `print()` с шуткой или интересным фактом о себе.

**Застрял?** Читай сообщение об ошибке как подсказку детектива:

- **`python is not recognized`** → Переустанови и отметь **Add to PATH**.
- **`No such file`** → Перейди в папку с файлом через `cd`.
- **Забыл кавычки** → Текст в `print()` должен быть в `"..."`.

## Уголок отладки

**Проблема:** Терминал пишет `'python' is not recognized as an internal or external command`.

**Причина:** Windows не находит Python. Часто не была отмечена галочка **Add to PATH**.

**Решение:** Запусти установщик снова → **Modify** → включи **Add Python to environment variables** → заверши установку → закрой и открой терминал → снова введи `python --version`.
