## What you'll learn / Что ты узнаешь

- `cd` and `cd ..` — walk into and out of folders
- `dir` / `ls` — list what is inside a folder
- Read the path shown in your terminal prompt
- `cls` / `clear` — clean the screen
- Tab key to autocomplete folder names
