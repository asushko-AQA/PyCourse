# Quest Paths — Treasure Hunt Checklist

Tick each box when you finish a step.

## Path A — PyCourse folder layout

- [ ] Open terminal in VS Code
- [ ] `cd` to your `PyCourse` folder (or project root)
- [ ] `cd course-1-python-basics\block-1-meeting-your-computer`
- [ ] `dir` — see lesson folders listed
- [ ] `cd lesson-1-1-installing-python`
- [ ] `dir` — find `starter` and `README.md`
- [ ] `cd ..` — go back one level
- [ ] Confirm you are in `block-1-meeting-your-computer` again

## Path B — Any computer (create your own maze)

- [ ] Create folder `quests\treasure` on your Desktop (or in your project)
- [ ] Create `clue.txt` inside with one sentence: "The treasure is in treasure.py"
- [ ] Open terminal and `cd` to Desktop (or parent folder)
- [ ] `cd quests\treasure`
- [ ] `dir` — see `clue.txt`
- [ ] Copy `treasure.py` from this lesson's `starter/` into this folder
- [ ] `python treasure.py` — see `Treasure found!`
