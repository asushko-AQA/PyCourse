## Объяснение

С возвращением, Создатель! На Уровне 1 ты открыл **Портал команд** (терминал) и сделал первые шаги. Теперь ты станешь **мастером лабиринта** — будешь ходить по папкам, как по комнатам в подземелье.

**Что уже было на Уровне 1:** открыть терминал, один пример `cd`, запуск `python hello.py`.

**Что нового на Уровне 2:** вернуться назад с `cd ..`, **посмотреть** файлы с `dir`, читать **текущий путь**, очищать экран.

Папки — это **комнаты**:

- `cd имя_папки` — войти **в** комнату
- `cd ..` — выйти **назад** в предыдущую комнату
- `dir` — осмотреться и увидеть, что внутри

---

### Шаг 1: Открой Портал команд

1. Открой VS Code с папкой проекта.
2. **Terminal → New Terminal** (или **Ctrl + `**).
3. Текст перед мигающим курсором — твой **текущий путь**, папка, в которой ты стоишь.

**Пример приглашения:**

```text
C:\Users\ТвоёИмя\Documents\PyCourse>
```

Путь перед `>` — твоё место в лабиринте.

---

### Шаг 2: Войти в папку с `cd`

`cd` — **c**hange **d**irectory, перейти в другую папку.

```text
cd Documents
```

**Ожидается:** приглашение обновится. Ты внутри `Documents`.

**Совет:** набери первые буквы имени папки и нажми **Tab** — терминал может дописать имя сам!

---

### Шаг 3: Выйти назад с `cd ..`

Две точки `..` означают «папка уровнем выше».

```text
cd ..
```

**Ожидается:** в приглашении родительская папка. Ты вернулся на один уровень.

---

### Шаг 4: Список файлов с `dir`

`dir` показывает содержимое текущей папки (Windows).

```text
dir
```

**Пример вывода (корень проекта):**

```text
 Directory of C:\Users\ТвоёИмя\Documents\PyCourse

course-1-python-basics
documents
README.md
...
```

**Внутри Блока 1** (после `cd course-1-python-basics\block-1-meeting-your-computer`):

```text
 Directory of ...\block-1-meeting-your-computer

lesson-1-1-installing-python
lesson-1-2-using-the-terminal
lesson-1-3-running-your-first-script
...
```

На **Mac/Linux** вместо `dir` используй `ls`.

Ищи файлы `.py` — это скрипты Python!

---

### Шаг 5: Очистить экран

Слишком много текста? Очисти портал:

```text
cls
```

На **Mac/Linux:** `clear`.

**Ожидается:** чистый экран терминала. Путь остался тем же.


**Ожидается:** приглашение заканчивается на `lesson-1-2-using-the-terminal>`.


**Ожидается:** видны `starter`, `solution`, `README.md` и другие файлы.

## Пример кода

### Свиток команд (терминал)

```text
# Войти в папку
cd Documents

# Выйти на уровень выше
cd ..

# Список файлов (Windows)
dir

# Список файлов (Mac/Linux)
# ls

# Очистить экран (Windows)
cls

# Очистить экран (Mac/Linux)
# clear
```

**Файл: [starter/treasure.py](starter/treasure.py)**

## Уголок отладки

**Проблема:** `The system cannot find the path specified`.

**Причина:** опечатка в имени папки или папки нет по этому пути.

**Решение:**

1. `cd ..` — вернись в знакомую папку.
2. `dir` — смотри **точные** имена.
3. `cd` с правильным именем из списка.
4. **Tab** после нескольких букв — меньше ошибок.
