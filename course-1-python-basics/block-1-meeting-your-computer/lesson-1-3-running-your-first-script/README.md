## What you'll build / Что ты создашь

**EN:** A reliable launch workflow — save `starter/launch.py` next to `hello.py` and run both with `python starter\hello.py` and `python starter\launch.py`.

**RU:** Надёжная последовательность запуска — сохранить `starter/launch.py` рядом с `hello.py` и запустить оба скрипта из папки `starter/`.

## What you'll learn / Что ты узнаешь

- Save vs Save As — keep the `.py` extension
- Explorer and terminal show the same folder
- `cd` to the script folder before `python file.py`
- VS Code Run button vs typing in the terminal
- Fix `can't open file` errors
