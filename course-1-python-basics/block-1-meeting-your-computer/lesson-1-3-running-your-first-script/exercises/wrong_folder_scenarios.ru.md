# Неправильная папка — почини сам!

Тренировка: найди и исправь проблемы запуска. Делай по порядку.

Все скрипты этого урока лежат в **`starter/`**.

## Сценарий 1 — Не та папка

1. Открой терминал в VS Code, когда в проводнике виден `starter/hello.py`.
2. Перейди в папку урока, затем `cd ..` дважды — ты **вне** папки урока.
3. Запусти:

```text
python starter\hello.py
```

**Что произошло?** Запиши текст ошибки.

**Исправление:** `cd` обратно в папку урока (где есть `starter/`). Запусти снова, пока не увидишь три строки.

- [ ] Я увидел ошибку
- [ ] Я исправил папку

## Сценарий 2 — Неправильное имя файла

1. Из **папки урока** запусти:

```text
python starter\helloo.py
```

(лишняя буква специально)

**Что произошло?**

**Исправление:** точное имя: `python starter\hello.py`

- [ ] Я увидел ошибку
- [ ] Я исправил имя

## Сценарий 3 — Скрытое расширение .txt

1. В **`starter/`** сохрани копию скрипта как `test.py`.
2. В проводнике: **Вид → Показать → Расширения имён файлов**.
3. Если файл `test.py.txt` — переименуй в `test.py`.
4. Из папки урока:

```text
python starter\test.py
```

- [ ] Файл заканчивается только на `.py`
- [ ] `python starter\test.py` работает
