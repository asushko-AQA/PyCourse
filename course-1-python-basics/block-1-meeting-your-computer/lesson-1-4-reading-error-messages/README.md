## What you'll learn / Что ты узнаешь

- Errors are clues, not failures
- Read the last line of a traceback first
- **SyntaxError** — broken grammar in your Python code
- **can't open file** — wrong folder or wrong filename
- Jump to line N in VS Code with Ctrl+G
