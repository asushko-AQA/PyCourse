## What you'll build / Что ты создашь

**EN:** Your own `my_mission/badge.py` — a 5–8 line victory script you create, navigate to, run, break, and fix.

**RU:** Свой `my_mission/badge.py` — победный скрипт на 5–8 строк: создать, найти папку, запустить, сломать и починить.

## What you'll learn / Что ты узнаешь

- Combine all Block 1 skills in one mission
- Create a new folder for your project
- Write and run a multi-line `print()` script
- Break your code on purpose and fix it
- Complete the Block 1 readiness checklist
