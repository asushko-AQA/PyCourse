## For parents and teachers

**Interactive lessons:** Lessons 2.2–2.5 wait for keyboard input in the terminal. Type answers and press Enter when the cursor blinks.

**Strings vs numbers:** Text goes in quotes (`"Alex"`); whole numbers do not (`11`). Lesson 2.3 introduces `int()` to convert typed text into numbers for math.

**Capstone folder:** Students create `my_data/` at the project root (next to `my_mission/`), mirroring Block 1's capstone pattern.
