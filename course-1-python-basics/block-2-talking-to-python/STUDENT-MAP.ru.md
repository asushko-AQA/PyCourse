## Путь B — нет полного репозитория?

| Урок | Вместо длинного `cd`… |

| 2.1 | Открой VS Code в любой папке; скопируй `starter/character_sheet.py` |
| 2.2 | Та же папка, что и скрипты 2.1, или любая твоя папка |
