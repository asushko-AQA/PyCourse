## What you'll build / Что ты создашь

**EN:** A `character_sheet.py` script that stores your name, age, and favorite game in variables and prints a labeled character sheet.

**RU:** Скрипт `character_sheet.py`, который хранит имя, возраст и любимую игру в переменных и печатает лист персонажа.

## What you'll learn / Что ты узнаешь

- What a variable is (a labeled magic box)
- Strings — text in quotes
- Integers — whole numbers without quotes
- Print variables with labels
- NameError when you typo a variable name

## Quick drills / Быстрые упражнения

1. Change `age = 11` to your real age — predict the output before you run.
2. Add a fourth variable `school = "Your School"` and one more `print()`.
3. Typo `name` as `nmae` on purpose — read the NameError, then fix it.
