### Шаг 1: Что такое переменная?

**Переменная** — подписанная коробка с именем. Ты кладёшь значение внутрь через `=`:

```python
name = "Alex"
```

- `name` — надпись на коробке
- `=` означает «положи это значение внутрь»
- `"Alex"` — значение (текст в кавычках)

---

### Шаг 2: Два вида коробок

**Строки** — текст в кавычках:

```python
favorite_game = "Minecraft"
```

**Целые числа** — целые числа **без** кавычек:

```python
age = 11
```

Если поставить кавычки вокруг числа (`"11"`), Python считает его текстом, а не числом для математики. Это исправим в Уроке 2.3.

### Шаг 6: Повторное использование переменной

Одну переменную можно использовать в нескольких `print()`:

```python
print("Name:", name)
print(name, "is ready for adventure!")
```

Измени значение в коробке один раз — каждый `print()` с `name` обновится автоматически.

## Уголок отладки

**Проблема:** `NameError: name 'nmae' is not defined`

**Причина:** Python ищет коробку с надписью `nmae`, а ты создал только `name`. Имена переменных должны совпадать **точно** — орфография и регистр.

**Исправление:** Найди опечатку. Верни `nmae` обратно в `name`. Запусти снова.

## Проверь себя

Выбери лучший ответ на каждый вопрос. Сначала попробуй без подсказок!

1. Что делает знак `=` в строке `name = "Alex"`?
   - **a)** Сравнивает два значения
   - **b)** Кладёт значение в подписанную переменную
   - **c)** Печатает текст на экран
   - **d)** Удаляет переменную

2. Какая строка правильно хранит **целое число**?
   - **a)** `age = "11"`
   - **b)** `age = 11`
   - **c)** `age = True`
   - **d)** `11 = age`

3. Если поставить кавычки вокруг числа (`"11"`), Python считает его…
   - **a)** Целым числом для математики
   - **b)** Текстом (строкой), а не числом для математики
   - **c)** Значением True/False
   - **d)** Ошибкой, которая всегда останавливает программу

4. Ошибка `NameError: name 'nmae' is not defined`. Вероятная причина?
   - **a)** Опечатка в имени переменной (орфография должна совпадать точно)
   - **b)** Забыли `input()`
   - **c)** Использовали `==` вместо `=`
   - **d)** Число было слишком большим

5. Ты меняешь `name = "Alex"` на `name = "Sam"`. Что произойдёт с каждым `print()`, где есть `name`?
   - **a)** Автоматически покажут новое значение
   - **b)** Всё ещё покажут "Alex", пока не перезагрузишь компьютер
   - **c)** Python удалит остальные переменные
   - **d)** Нужно переименовать каждую строку `print()` вручную
