## What you'll build / Что ты создашь

**EN:** An interactive `greeting.py` that asks your name and age, then responds with personalized f-string messages.

**RU:** Интерактивный `greeting.py`, который спрашивает имя и возраст и отвечает персональными f-строками.

## What you'll learn / Что ты узнаешь

- `input()` — ask questions in the terminal
- f-strings — fill-in-the-blank scrolls with `{variable}`
- `input()` always returns a **string** (foreshadow for Lesson 2.3)

## Quick drills / Быстрые упражнения

1. Remove the `f` from one f-string — predict what prints. Run to confirm.
2. Add a third `input()` for favorite color.
3. Type your answers slowly — watch the terminal wait for each question.
