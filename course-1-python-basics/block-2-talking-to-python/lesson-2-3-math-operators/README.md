## What you'll build / Что ты создашь

**EN:** A `calculator.py` that asks for two numbers and prints sum, difference, product, and quotient.

**RU:** `calculator.py`, который спрашивает два числа и печатает сумму, разность, произведение и частное.

## What you'll learn / Что ты узнаешь

- Math operators: `+`, `-`, `*`, `/`
- `int()` to convert input strings to numbers
- `ValueError` when input is not a number

1. Predict `10 / 3` before running — write your guess first.
2. Run with `10` and `2` — check all four results.
3. Try typing `abc` — read the friendly `ValueError`.
