## Объяснение

Время **заклинаний калькулятора**! Python умеет складывать, вычитать, умножать и делить — но помни: `input()` даёт **текст**. Оберни в `int()`, чтобы превратить цифры в числа.

| Оператор | Значение | Пример |
|----------|----------|--------|
| `+` | Сложение | `5 + 3` → `8` |
| `-` | Вычитание | `5 - 3` → `2` |
| `*` | Умножение | `5 * 3` → `15` |
| `/` | Деление | `10 / 4` → `2.5` |

**Заметка:** `/` даёт десятичное число при необходимости. `//` и `%` — позже.

### Шаг 1: Преобразуй ввод через int()

```python
num1 = int(input("Enter the first number: "))
```

Ты вводишь `7`, жмёшь Enter — в `num1` попадает **число** 7, а не текст `"7"`.

## Пример кода

```python
num1 = int(input("Enter the first number: "))
num2 = int(input("Enter the second number: "))

print("Sum:", num1 + num2)
print("Difference:", num1 - num2)
print("Product:", num1 * num2)
print("Quotient:", num1 / num2)
```

## Уголок отладки

**Проблема:** `ValueError: invalid literal for int() with base 10: 'hello'`

**Причина:** `int()` ожидал цифры, а получил текст. `input()` всегда возвращает строку — если это не целое число, преобразование падает.

**Исправление:** Вводи только числа вроде `5` или `12`. Без букв и десятичных пока.
