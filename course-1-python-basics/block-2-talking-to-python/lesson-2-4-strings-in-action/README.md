## What you'll build / Что ты создашь

**EN:** A Mad-Libs style `madlibs.py` that asks for silly words and prints a story with `.upper()`, `.lower()`, and slicing.

**RU:** Mad-Libs `madlibs.py` — смешные слова и история с `.upper()`, `.lower()` и срезами.

## What you'll learn / Что ты узнаешь

- `.upper()` and `.lower()` — text transformer machine
- Slicing `[0:3]` — cut a piece from a string
- Building stories from multiple `input()` calls

## Quick drills / Быстрые упражнения

1. Predict `"hello".upper()` before typing it in Python.
2. Run madlibs with sample input: hero `Sam`, creature `dragon`, place `castle`, power `laser`.
3. Change the creature slice from `[0:3]` to `[0:2]` — see the difference.
