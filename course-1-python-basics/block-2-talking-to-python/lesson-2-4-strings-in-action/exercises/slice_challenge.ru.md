# Задание на срезы (бонус) — Урок 2.4

> [English →](slice_challenge.md)

Срез вырезает кусок строки. Формат: `word[start:end]` — включает `start`, останавливается **перед** `end`.

---

## Разминка

```python
word = "Minecraft"
print(word[0:3])
```

**Твой прогноз:** _______  
**Ответ:** `Min` (буквы с индексами 0, 1, 2)

---

```python
hero = "Alex"

```

## Задание 2 — IndexError (дополнительно)

```python
short = "Hi"
print(short[0:10])
```

В Python это работает — останавливается в конце строки. Вывод: `Hi`

А это падает:

```python
print(short[5])
```

**Ошибка:** `IndexError: string index out of range`  
**Причина:** Индекс 5 не существует в двухбуквенной строке.
