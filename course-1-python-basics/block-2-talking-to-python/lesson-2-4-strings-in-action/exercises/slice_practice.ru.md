# Практика срезов — бонусный квест

Срез берёт часть строки: `word[0:3]` значит «с индекса 0 до (не включая) 3».

## Безопасные срезы

```python
game = "Minecraft"
print(game[0:3])   # Min
print(game[0:4])   # Mine
```

## Бонус — IndexError

Что будет, если срезать за пределами?

```python
word = "Hi"
print(word[0:10])  # Попробуй — упадёт?
```

**Ответ:** Python прощает — даст `"Hi"`, без ошибки. Но **пустой** срез на одной букве может удивить:

```python
letter = "A"
print(letter[5:10])  # Что напечатает?
```

**Ответ:** Пустая строка `""` — без падения, просто ничего!
