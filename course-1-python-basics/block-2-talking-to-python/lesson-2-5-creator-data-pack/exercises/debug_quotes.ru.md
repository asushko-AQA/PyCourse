# Задание: сломанные кавычки — Урок 2.5

> [English →](debug_quotes.md)

Потренируйся чинить сломанные строки — тот же навык, что нужен в итоговом квесте.

---

## Сломанный код 1

```python
print("=== CREATOR DATA PACK ===)
```

**Ошибка:** `SyntaxError` — незакрытая строка  
**Исправление:** Добавь закрывающую кавычку: `"=== CREATOR DATA PACK ==="`

---

## Сломанный код 2

```python
name = input("What is your name? )
```

**Ошибка:** `SyntaxError` — нет `"` перед `)`  
**Исправление:** `name = input("What is your name? ")`
