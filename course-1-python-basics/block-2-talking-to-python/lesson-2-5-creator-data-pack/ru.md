## Уголок отладки

**Проблема:** `TypeError: can't multiply sequence by non-int of type 'str'`

**Причина:** `age = input(...)` без `int()`, потом `age * 365`. Python считает возраст текстом.

**Исправление:** `age = int(input("How old are you? "))` — преобразуй перед математикой.
