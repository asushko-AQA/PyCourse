## What you'll build / Что ты создашь

**EN:** `true_false_quiz.py` — a script that prints labeled True/False results for comparisons.

**RU:** `true_false_quiz.py` — скрипт, который печатает результаты сравнений (True/False).

## What you'll learn / Что ты узнаешь

- `True` and `False` — quest pass / quest fail stamps
- Comparison operators: `==`, `!=`, `>`, `<`, `>=`, `<=`
- Combining comparisons with `and` and `or`

## Quick drills / Быстрые упражнения

1. Before running, predict each line — write True or False on paper.
2. Change `5 == 3` to `5 == 5` — predict, then run.
3. Add `print("7 <= 7:", 7 <= 7)` at the bottom.
