## Объяснение

В уроке 3.1 ты **печатал** True/False. Теперь ты **действуешь** с помощью **`if`** — развилка на тропе.

| Ключевое слово | Когда выполняется |
|----------------|-------------------|
| `if` | Первая проверка |
| `elif` | Иначе если — следующая проверка |
| `else` | Если ничего не подошло |

**Правила:**

1. **Двоеточие** `:` в конце каждой строки `if` / `elif` / `else`
2. **Отступ** (4 пробела) у кода внутри ветки

```python
if choice == "left":
    print("Кристалл найден!")
elif choice == "right":
    print("Робот-проводник!")
else:
    print("Перекус!")
```

Выполняется **только одна** ветка — первая подходящая.

## Пример кода

```python
choice = input("Go left or right? (left/right): ")

if choice == "left":
    print("You find a glowing crystal! +10 quest points.")
elif choice == "right":
    print("You meet a friendly robot guide. It waves hello.")
else:
    print("You sit down for a snack. Adventure can wait!")
```
