# Loop Drills — Extra Challenges

**Lesson:** 3.3 · **Block:** Making Choices

Try these after completing `countdown.py` and `times_table.py`.

---

## Drill 1: Count up

Write a new script `count_up.py` that prints `1` through `5`, then `"Go!"`.

**Hint:** Start `counter = 1` and use `while counter <= 5:`.

---

## Drill 2: Fives table

Copy `times_table.py` and change it to print the **5 times table** (5 x 1 through 5 x 12).

---

## Drill 3: Sum loop

Use a `for` loop and `range(1, 6)` to print the sum `1 + 2 + 3 + 4 + 5 = 15`.

**Hint:** Use a variable `total = 0` and add `i` each loop.
