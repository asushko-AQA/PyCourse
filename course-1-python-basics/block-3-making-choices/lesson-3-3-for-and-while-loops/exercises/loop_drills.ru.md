## Задача 1: Счёт вверх

Напиши `count_up.py`, который печатает `1`–`5`, затем `"Go!"`.

**Подсказка:** `counter = 1`, `while counter <= 5:`.

## Задача 3: Сумма в цикле

С помощью `for` и `range(1, 6)` выведи сумму `1 + 2 + 3 + 4 + 5 = 15`.

**Подсказка:** `total = 0`, каждый раз `total = total + i`.
