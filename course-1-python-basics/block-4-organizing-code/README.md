## For parents and teachers

**One concept per lesson:** Functions (4.1), lists (4.2), and dicts (4.3) are taught separately before the capstone combines them.

**Interactive lessons:** Lessons 4.1 and 4.4 wait for keyboard input. Type answers and press Enter when the cursor blinks.

**Capstone folder:** Students create `my_adventure/` at the project root (next to `my_data/`), mirroring Block 2's capstone pattern.
