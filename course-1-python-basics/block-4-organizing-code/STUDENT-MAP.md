## Path B — no full repo?

| Lesson | Instead of long `cd`… |
|--------|------------------------|
| 4.1 | Copy `helpers.py` anywhere; `cd` to that folder before running |
| 4.2 | Copy `inventory.py`; run from the folder where you saved it |
| 4.3 | Copy `character_stats.py`; run from your project folder |
| 4.4 | Create `my_adventure` on Desktop or inside your project — same idea as `my_data` |
