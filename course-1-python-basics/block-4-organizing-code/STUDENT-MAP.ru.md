## Путь B — нет полного репозитория?

| Урок | Вместо длинного `cd`… |
|------|------------------------|
| 4.1 | Скопируй `helpers.py` куда угодно; `cd` в папку перед запуском |
| 4.2 | Скопируй `inventory.py`; запускай из папки, где сохранил |
| 4.3 | Скопируй `character_stats.py`; запускай из своей папки проекта |
| 4.4 | Создай `my_adventure` на Рабочем столе или в проекте — как `my_data` |
