## What you'll build / Что ты создашь

**EN:** A `helpers.py` script with `draw_banner()` and `ask_yes_no()` functions that `main()` calls to greet you and ask a yes/no question.

**RU:** Скрипт `helpers.py` с функциями `draw_banner()` и `ask_yes_no()`, которые `main()` вызывает для приветствия и вопроса да/нет.

## What you'll learn / Что ты узнаешь

- What a function is (a reusable recipe)
- Writing functions with `def`
- Calling a function by name with `()`
- Using `return` to send a value back
- Why you must **call** a function for it to run

## Quick drills / Быстрые упражнения

1. Change the banner text to your own workshop name — predict the output before you run.
2. Add a third function `draw_line()` that prints ten dashes, and call it from `main()`.
3. Comment out the line `main()` at the bottom — run. Nothing happens! Uncomment it.
