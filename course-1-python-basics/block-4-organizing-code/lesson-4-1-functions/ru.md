## Объяснение

Добро пожаловать в **Мастерскую приключений**! В Блоке 3 ты писал ветки `if` и циклы. Теперь ты изучишь **функции** — многоразовые рецепты, которые можно запускать снова и снова без переписывания одного и того же кода.

**Функция** — именованный блок шагов. Ты **определяешь** её один раз через `def`, затем **вызываешь**, когда нужны эти шаги:

```python
def draw_banner(text):
    print("===")
    print(text)
    print("===")
```

Вызов `draw_banner("Привет")` выполняет все три строки `print()` внутри функции.

## Пример кода

**Файл: [starter/helpers.py](starter/helpers.py)**

```python
def draw_banner(text):
    print("===")
    print(text)
    print("===")


def ask_yes_no(question):
    answer = input(f"{question} (yes/no): ").lower().strip()
    if answer in ("y", "yes"):
        return "yes"
    return "no"


def main():
    draw_banner("Adventure Workshop")
    answer = ask_yes_no("Ready to build helper functions?")
    print(f"You answered: {answer}")


main()
```

## Отладка

**Проблема:** Ты написал `draw_banner("Adventure Workshop")` внутри `def main()`, но забыл **вызвать** `main()` в конце файла. Скрипт запускается и завершается без вывода.

**Причина:** Определение функции только **сохраняет** рецепт. Python не выполняет шаги, пока ты не **вызовешь** функцию по имени: `main()`.

**Исправление:** Убедись, что последняя строка файла — `main()` (без отступа — на верхнем уровне). Запусти снова.
