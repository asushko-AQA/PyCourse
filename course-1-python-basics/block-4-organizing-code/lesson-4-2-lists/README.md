## What you'll build / Что ты создашь

**EN:** An `inventory.py` script that stores game items in a list, adds more with `.append()`, and prints a numbered inventory with a `for` loop.

**RU:** Скрипт `inventory.py`, который хранит предметы в списке, добавляет новые через `.append()` и печатает нумерованный инвентарь циклом `for`.

## What you'll learn / Что ты узнаешь

- What a list is (an ordered backpack of items)
- Creating a list with `[ ]`
- Adding items with `.append()`
- Looping with `for` and `range(len(...))`
- List indexes start at `0` (but humans count from 1)

## Quick drills / Быстрые упражнения

1. Add `"map"` with `.append()` — predict the new item count before you run.
2. Print only the first item using `inventory[0]`.
3. Change the loop to start numbering at 0 — notice how it looks to a player.
