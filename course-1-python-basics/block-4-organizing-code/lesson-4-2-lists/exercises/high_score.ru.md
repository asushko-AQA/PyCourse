# Таблица рекордов — дополнение к Уроку 4.2

> [English →](high_score.md)

Создай **таблицу рекордов** с помощью списка — функции пока не нужны.

---

## Цель

Создай `high_scores.py` (в любой папке), который:

1. Начинает со списка из трёх очков: `[120, 85, 200]`
2. Спрашивает новый результат через `input()` и `int()`
3. Добавляет очко через `.append()`
4. Печатает каждый результат на отдельной строке с номером (`1.`, `2.`, …)

---

## Пример запуска

```text
=== HIGH SCORES ===
Enter your score: 150
1. 120
2. 85
3. 200
4. 150
```

---

## Подсказки

- Преобразуй ввод: `new_score = int(input("Enter your score: "))`
- Используй тот же шаблон `for i in range(len(scores)):` из [inventory.py](../starter/inventory.py)
- **Застрял на сортировке?** Бонус — попробуй `scores.sort(reverse=True)` после базовой версии

---

## Бонусные звёзды

Напечатай **максимальный** результат после списка (подсказка: пройди циклом и запомни наибольшее, или изучи `max(scores)`).
