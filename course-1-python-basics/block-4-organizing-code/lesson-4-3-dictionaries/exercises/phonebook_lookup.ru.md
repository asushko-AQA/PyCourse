# Поиск в телефонной книге — дополнение к Уроку 4.3

> [English →](phonebook_lookup.md)

Создай мини-**телефонную книгу** со словарём имён и номеров.

---

## Цель

Создай `phonebook.py`, который:

1. Хранит минимум три контакта в словаре — ключи имена, значения строки с номерами
2. Спрашивает `Whose number? ` через `input()`
3. Использует `.get()`, чтобы напечатать номер или `"Not found"`, если имени нет

---

## Пример словаря

```python
phonebook = {
    "Alex": "555-0100",
    "Sam": "555-0200",
    "Jordan": "555-0300",
}
```

---

## Пример запуска

```text
=== PHONEBOOK ===
Whose number? Sam
Number: 555-0200
```

Второй запуск:

```text
Whose number? Riley
Number: Not found
```

---

## Подсказки

- Поиск: `number = phonebook.get(name, "Not found")`
- Используй `.strip()` для ввода, чтобы пробелы не ломали поиск
- **Бонус:** Пройди циклом по всем контактам и напечатай `Name: number` на отдельных строках
