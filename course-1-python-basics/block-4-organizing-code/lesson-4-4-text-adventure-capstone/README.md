## What you'll build / Что ты создашь

**EN:** A room-based text adventure in `my_adventure/game.py` — functions, a list inventory, dict rooms, and a win condition when you open the garden chest with the golden key.

**RU:** Текстовое приключение по комнатам в `my_adventure/game.py` — функции, список-инвентарь, словарь комнат и победа при открытии сундука ключом из библиотеки.

## What you'll learn / Что ты узнаешь

- Combining functions, lists, and dicts in one project
- Nested dicts (rooms inside `rooms`)
- A game loop with `while True` and `break`
- Creating `my_adventure/` at the project root
- Win conditions that check room + inventory
