# Идеи для расширения — Урок 4.4

> [English →](extension_prompts.md)

Закончил базовое приключение? Прокачай игру одной или несколькими идеями ниже.

---

## Новые команды

| Команда | Идея |
|---------|------|
| `look` | Снова напечатать описание комнаты без перехода |
| `help` | Вывести список команд ещё раз |
| `drop sword` | Убрать предмет из инвентаря обратно в комнату |

---

## Больше комнат

Добавь комнату `"cave"` из `"garden"` через `"north": "cave"`. Положи туда `"magic gem"` и требуй **и** ключ, **и** камень для победы.

---

## Очки здоровья

Храни `"hp": 100` в словаре игрока. Вычитай 10 HP в комнате `"trap"`. Game over при HP = 0.

---

## Случайная встреча

```python
import random

if random.randint(1, 3) == 1:
    print("A friendly fox blocks the path!")
```

Используй редко — одной комнаты-сюрприза достаточно.

---

## Поделись миром

Нарисуй карту на бумаге с id комнат (`start`, `library`, …), чтобы друг мог играть без чтения кода.
