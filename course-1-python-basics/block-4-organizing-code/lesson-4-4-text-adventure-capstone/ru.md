### Шаг 1: Пойми словарь rooms

Каждая комната — словарь внутри большого `rooms`:

```python
"library": {
    "name": "Scroll Library",
    "description": "Dusty scrolls...",
    "south": "start",
    "item": "golden key",
}
```

- `"name"` и `"description"` — для вывода на экран
- `"south": "start"` — выход, ключ направления указывает на id другой комнаты
- `"item"` — необязательная добыча в комнате

### Шаг 3: Игровой цикл

`main()` запускает цикл `while True`:

1. Показать текущую комнату
2. Прочитать команду через `input()`
3. Обработать `north`/`south`/…, `take`, `inventory`, `open`, `quit`
4. `break` при победе или выходе

## Отладка

**Проблема:** `TypeError` или неверная комната, когда пишешь `rooms["north"]` вместо `rooms[current_room]["north"]`.

**Причина:** Ты смешал **индекс списка** со **ключом словаря**. `rooms` — словарь с ключами id комнат (`"start"`, `"library"`). Направления вроде `"north"` живут **внутри** словаря каждой комнаты — не на верхнем уровне.

**Исправление:** Сначала получи текущую комнату: `room = rooms[current_room]`. Затем проверь `if direction in room:` перед переходом. Не используй числовой индекс для `rooms`, если специально не строишь список (в этой игре только ключи словаря).
