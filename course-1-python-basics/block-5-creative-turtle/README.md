## For parents and teachers

**Turtle window:** A separate drawing window opens when the script runs. It may appear **behind** VS Code — check the taskbar for a Python or Turtle icon.

**Keep the window open:** Every turtle script should end with `turtle.done()`. Without it, the window flashes closed and students think the program failed.

**Stopping a script:** Close the turtle window, or press **Ctrl+C** in the VS Code terminal if the script will not exit.

**No pip install:** `turtle` ships with Python 3 on Windows. If `import turtle` fails, reinstall Python and enable "tcl/tk and IDLE" during setup.

**Prerequisites:** Students should be comfortable with loops (Block 3) and functions (Block 4) before Lesson 5.3.

**Block 5 capstone:** Lesson 5.3 includes **`my_gallery/gallery.py`**. See [Course 1 graduation checklist](../STUDENT-MAP.md#course-1-graduation-checklist).
