## What you'll build / Что ты создашь

**EN:** A `star.py` script that uses a loop and changing `pencolor()` values to draw a colorful five-point star.

**RU:** Скрипт `star.py`, который с помощью цикла и смены `pencolor()` рисует цветную пятиконечную звезду.

## What you'll learn / Что ты узнаешь

- Combine a `for` loop with turtle drawing commands
- Change line color with `pencolor("red")` and a list of color names
- Draw a star with `forward()` and `right(144)`
- Find the turtle window when it opens behind VS Code

## Quick drills / Быстрые упражнения

1. Add `"purple"` to the colors list — run and count the star points.
2. Change `right(144)` to `right(72)` — predict the new shape.
3. Set `t.speed(1)` for slow motion, then `t.speed(0)` for instant drawing.
