## Объяснение

Пора добавить **цвет** в Студию Turtle! В уроке 5.1 ты рисовал фигуры в цикле. Теперь ты нарисуешь **пятиконечную звезду**, меняя цвет пера на каждой вершине.

| Команда | Что делает |
|---------|------------|
| `t.pencolor("red")` | Цвет линии (имя цвета **в кавычках**) |
| `t.speed(0)` | Максимальная скорость рисования |
| `t.right(144)` | Поворот на 144° — «магический» угол для звезды |

**Почему 144?** Для пятиконечной звезды нужны повороты на 144° (не 72°). Пока доверься числу — в упражнениях попробуй изменить и посмотри, что сломается!

Храни цвета в **списке** и выбирай по одному за цикл:

```python
colors = ["red", "orange", "yellow", "green", "blue"]
t.pencolor(colors[point])
```

## Пример кода

**Файл: [starter/star.py](starter/star.py)**

```python
import turtle

t = turtle.Turtle()
t.speed(0)

colors = ["red", "orange", "yellow", "green", "blue"]

for point in range(5):
    t.pencolor(colors[point])
    t.forward(100)
    t.right(144)

turtle.done()
```

## Уголок отладки

**Проблема:** `turtle.TurtleGraphicsError: bad color string: purpel`

**Причина:** `pencolor()` принимает **правильное имя цвета**. `"purpel"` — опечатка, Python не знает такого слова.

**Исправление:** Пиши имена цветов аккуратно. Частые: `"red"`, `"blue"`, `"green"`, `"orange"`, `"purple"`, `"black"`, `"white"`. Исправь опечатку и запусти снова.

**Ещё проверь:** Если ничего не видно, окно может быть **за** VS Code — нажми на иконку Python на панели задач.
