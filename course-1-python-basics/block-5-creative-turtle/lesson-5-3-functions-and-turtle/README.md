## What you'll build / Что ты создашь

**EN:** A `snowflake.py` script with a reusable `draw_branch(length)` function — plus **`my_gallery/gallery.py`** combining your star and snowflake.

**RU:** `snowflake.py` с функцией `draw_branch` — плюс **`my_gallery/gallery.py`** (звезда и снежинка вместе).

## What you'll learn / Что ты узнаешь

- Define a function with `def` that uses the turtle
- Pass a number parameter (`length`) into your function
- Call the same function twice with `left()` turns between calls
- Indent code correctly inside a function body
