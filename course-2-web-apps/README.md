# Course 2: Web Applications with Python

> **Audience:** Ages 11+ · **Python:** 3.12+ · **Flask** + HTML templates  
> **Pace:** ~4–6 weeks at 1–2 lessons/week (~30–45 min each)  
> **Prerequisites:** Complete [Course 1](../course-1-python-basics/README.md) — especially functions, lists, dicts, and the [Block 5 readiness checklist](../course-1-python-basics/block-5-creative-turtle/README.md#block-5-readiness-checklist).

## Exit skills

When you finish Block 2, you can:

- Create and activate a Python virtual environment
- Install packages with `pip` and run a local Flask dev server
- Define routes, dynamic URLs, and multi-page mini-sites
- Use Jinja2 templates, HTML forms (GET/POST), static CSS, and flash messages
- Build two capstone web apps: Mad-Libs and a calculator
