## What you'll build / Что ты создашь

**EN:** A terminal quiz (`web_quiz.py`) that asks about browser, server, and URL — plus a printed request/response demo showing how a page travels to you.

**RU:** Терминальный квиз (`web_quiz.py`) про браузер, сервер и URL — и демонстрация запроса/ответа, показывающая путь страницы к тебе.

## What you'll learn / Что ты узнаешь

- What a **browser** does (opens pages, sends requests)
- What a **server** does (waits for requests, sends responses)
- What a **URL** is (the web address you type)
- How **request** and **response** travel between browser and server

## Quick drills / Быстрые упражнения

1. Type a real URL in your browser — watch the address bar. That string is the URL.
2. Run the solution quiz and score 3/3 before building your own version.
3. Read the request/response demo at the end — say each line out loud as "browser" or "server".
