## What you'll build / Что ты создашь

**EN:** A short `check_flask.py` script that imports Flask inside your activated venv and prints the version number.

**RU:** Короткий скрипт `check_flask.py`, который импортирует Flask в активированном venv и выводит номер версии.

## What you'll learn / Что ты узнаешь

- Activate your virtual environment before web work (quick reminder)
- Confirm Flask is installed with `pip list`
- Import Flask in Python and read its version

## Quick drills / Быстрые упражнения

1. Activate `.venv`, run `pip list` — find `Flask` in the list.
2. Run `python starter\check_flask.py` after finishing the TODOs.
3. Deactivate, run again — notice the import error (then reactivate!).
