## Объяснение

В Уроке 1.2 страница всегда показывала один и тот же текст. Настоящие сайты меняются по URL — например `/profile/Alex` показывает страницу Алекса.

**Динамический маршрут** имеет **переменную часть** в пути. Flask забирает эту часть и передаёт в функцию как аргумент — как будто вручает бейдж с именем гостю у входа.

### Шаг 1: Открой стартовое приложение

2. Открой [starter/app.py](starter/app.py).

Главный маршрут уже готов. Добавь один новый с плейсхолдером `<name>`.

---

### Шаг 2: Добавь `/hello/<name>`

Выполни TODO:

```python
@app.route("/hello/<name>")
def hello(name):
    return f"Hello, {name}!"
```

**Что означает каждая часть:**

| Часть | Значение |
|-------|----------|
| `/hello/<name>` | Путь со слотом для переменной `name` |
| `def hello(name):` | Flask подставляет `name` из URL |
| `f"Hello, {name}!"` | f-string — вставляет значение из URL в текст |

## Уголок отладки

**Проблема:** `TypeError` или странная ошибка про аргументы функции

**Причина:** Имя в `<name>` не совпадает с параметром функции или опечатка в декораторе.

**Решение:** `@app.route("/hello/<name>")` и `def hello(name):` — имена должны совпадать точно.

---

**Проблема:** `404 Not Found` для `/hello/Alex`

**Причина:** Сервер не запущен, неверный порт или маршрут не сохранён.

**Решение:** Убедись, что `python starter\app.py` работает, проверь написание (`/hello/`, не `/hellos/`), сохрани файл и перезапусти при необходимости.
