## Объяснение

Одно Flask-приложение может отдавать **много страниц** — каждый `@app.route` это ещё одна «дверь». Ты уже использовал строковые переменные (`<name>`). Теперь — **целочисленные** (`<int:a>`): Flask превращает части URL в числа, которые можно складывать.

Это **итог Блока 1**: маленький сайт со ссылками и калькулятор, читающий числа из адреса. В Блоке 2 будет калькулятор с формой — та же идея, красивее интерфейс.

---

### Шаг 1: Изучи главную страницу в стартере

Открой [starter/app.py](starter/app.py). Маршрут `/` уже возвращает HTML со ссылками:

- **About** → `/about`
- **Jokes** → `/jokes`
- **Примеры калькулятора** → `/add/3/5`

Теги **HTML** (HyperText Markup Language), которые встретишь:

| Тег | Назначение |
|-----|------------|
| `<h1>...</h1>` | Крупный заголовок |
| `<p>...</p>` | Абзац |
| `<a href='/about'>About</a>` | Кликабельная ссылка |

---

### Шаг 2: Добавь маршрут About

```python
@app.route("/about")
def about():
    return "<h1>About Me</h1><p>I am learning Flask and building web pages with Python!</p>"
```

Измени абзац — укажи **своё** имя или хобби.

---

### Шаг 3: Добавь маршрут Jokes

```python
@app.route("/jokes")
def jokes():
    return (
        "<h1>Jokes</h1>"
        "<p>Why do programmers prefer dark mode?</p>"
        "<p>Because light attracts bugs!</p>"
    )
```

Можешь вставить свой шутку — дружелюбную!

---

### Шаг 4: Добавь URL-калькулятор

```python
@app.route("/add/<int:a>/<int:b>")
def add(a, b):
    total = a + b
    return f"<h1>Calculator</h1><p>{a} + {b} = {total}</p>"
```

**`<int:a>`** говорит Flask: «это целое число, назови его `a`».

Для `/add/3/5` получается `a=3`, `b=5`, `total=8`.

### Шаг 5: Запусти и пройди по сайту

```text
cd course-2-web-apps\block-1-web-basics-flask\lesson-1-4-multiple-routes
python starter\app.py
```

Открой `http://127.0.0.1:5000` и:

## Уголок отладки

**Проблема:** Ссылка даёт `404 Not Found`

**Причина:** Нет функции маршрута, опечатка или сервер не перезапущен после сохранения.

**Решение:** Проверь, что `@app.route("/about")` совпадает с `href`. Сохрани, перезапусти `python starter\app.py`.

---

**Проблема:** `/add/three/five` не работает

**Причина:** `<int:a>` принимает только целые числа, не слова.

**Решение:** Используй цифры: `/add/3/5`. Поэтому это **URL-калькулятор** — числа живут в адресной строке.

---

**Проблема:** На странице видны сырые HTML-теги как текст

**Причина:** Редко во Flask — обычно опечатка в кавычках или случайный кортеж в return.

**Решение:** Возвращай одну строку: `return "<h1>Title</h1><p>Text</p>"` — проверь кавычки.
