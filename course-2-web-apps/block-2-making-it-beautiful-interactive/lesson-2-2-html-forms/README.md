## What you'll build / Что ты создашь

**EN:** A page with two forms that greet you by name — one uses GET (name in the URL), one uses POST (name hidden).

**RU:** Страница с двумя формами приветствия — GET (имя в URL) и POST (имя скрыто).

## What you'll learn / Что ты узнаешь

- HTML `<form>`, `<input>`, and `<button>`
- GET — data in `request.args`
- POST — data in `request.form`
