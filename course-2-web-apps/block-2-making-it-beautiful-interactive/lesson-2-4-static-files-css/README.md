## What you'll build / Что ты создашь

**EN:** A styled mini-site with CSS in `static/style.css`, linked via `url_for('static', ...)`.

**RU:** Стилизованный мини-сайт: CSS в `static/style.css`, подключение через `url_for('static', ...)`.

## What you'll learn / Что ты узнаешь

- Flask `static/` folder for CSS, images, and fonts
- `url_for('static', filename='style.css')` in templates
- Simple responsive layout with `@media` queries
