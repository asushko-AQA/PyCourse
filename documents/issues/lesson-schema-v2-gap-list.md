# Issue: Lesson schema v2 migration gap list (feeds plan 15)

**Status:** open
**Date:** 2026-06-24
**Severity:** medium
**Location:** all existing lessons (Course 1: 21, Course 2: 11)

## Problem

[Lesson schema v2](../plans/lesson-schema-v2.md) makes `## Code Example` and
`## Code Execution` **required** and standardizes `## Practice Task` as the canonical
exercise heading. Some published lessons predate this and need migration. This is the
authoritative per-block gap list consumed by **plan 15 — lesson-migration-batches**.

Generated by scanning every `en.md` for `## Code Example`, `## Code Execution`, and the
Practice Task / Try it yourself heading. `npm run validate-content` reports the same gaps
as non-fatal warnings.

## Expected

Every lesson has both code sections and uses `## Practice Task` (with localized RU
heading), EN/RU in parity.
