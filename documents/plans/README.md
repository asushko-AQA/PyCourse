## Status convention

| Status | Meaning |
|--------|---------|
| `draft` | Written, not yet peer-reviewed |
| `in_review` | Review agent has run; amendments pending |
| `approved` | Review complete; ready to implement |
| `implemented` | All done criteria met |

## Adding a new plan

1. Copy [_template.md](_template.md).
2. Set `**Status:** draft` and fill in Goal, Steps, Done criteria.
3. Add a row to the table above.
4. After review: `in_review` → merge amendments → `approved`.
5. After implementation: tick done criteria → `implemented`.
